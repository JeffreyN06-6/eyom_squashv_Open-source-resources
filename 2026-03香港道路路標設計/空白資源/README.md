## 💾 圖片命名規則 (Image Naming Convention)

🗂️本資料夾內的圖片遵循以下關鍵字命名規則：

| 關鍵字 | 定義 | 說明 |
| :--- | :--- | :--- |
| **Overhead** | 龍門架類 | 指跨越車道的大型標誌架結構 |
| **Post-mounted** | 路側路牌 | 指路側柱子上的標志 |
| **Green** | 綠色底面 | 標誌背景色為綠色 (快速公路) |
| **Blue** | 藍色底面 | 標誌背景色為藍色 (非快速公路) |
| **Arrow** | 包含箭頭 | 標誌內含有方向指示箭頭 |
| **NO** | 數字編號 | 標誌內包含幹線或出口編號或快速公路標志 |
| **C** | 標誌位置 | 位於上側的標誌 (Additional / Upper) |
| **B** | 標誌位置 | 使用虛線分開的標志(dotted line) |
| **A** | 標誌位置 | 位於下側的標誌 (Center / Lower) |

### 📝 命名範例
`Overhead_Green_Arrow_NO_CA1.png`  
（含意：上下各一個綠色底的龍門架標誌，且同時帶有箭頭及編號））
